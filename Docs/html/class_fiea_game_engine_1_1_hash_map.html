Detailed Description

template<typename TKey, typename TValue, typename HashFunctor = DefaultHash<TKey>, typename EqualityFunctor = DefaultEquality<TKey>>
class FieaGameEngine::HashMap< TKey, TValue, HashFunctor, EqualityFunctor >

Template HashMap class that maps TKeys to TValues and offers best case constant time lookups

Template Parameters
TKeyRequired type parameter that defines the type of the keys stored in the HashMap
TValueRequired type parameter that defines the type of the values store in the HashMap
HashFunctorOptional type parameter that defines what hash function is used to hash keys
EqualityFunctorOptional type parameter that defines how determine equality for keys
